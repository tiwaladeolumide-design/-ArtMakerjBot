# 🎨 ArtMakerjBot

An AI-powered Telegram bot for generating images from text descriptions.

## ✨ Features

- Multiple AI models (Replicate, OpenAI, Hugging Face)
- High-quality image generation
- Rate limiting (5 images per day)
- User statistics tracking
- Inline keyboard interactions
- Feedback system
- Auto-fallback between models
